## ಪೀಠಿಕೆ

ನಮ್ಮ ವೆಬ್‌ಸೈಟ್ ಅನ್ನು ನಿರ್ಮಿಸಲು ಮತ್ತು ನಿರ್ವಹಿಸಲು ಒಂದು ಆಧುನಿಕ **JAMstack** ವ್ಯವಸ್ಥೆಯನ್ನು ಬಳಸಲಾಗಿದೆ. ಇದರಲ್ಲಿ ಮೂರು ಪ್ರಮುಖ ಭಾಗಗಳಿವೆ:

- **Hugo** — ಸ್ಥಿರ ತಾಣ ಜನರೇಟರ್ (Static Site Generator)
- **Sveltia CMS** — ಲೇಖನಗಳನ್ನು ಸಂಪಾದಿಸಲು ಬ್ರೌಸರ್ ಆಧಾರಿತ ಆಡಳಿತ ಪ್ಯಾನೆಲ್
- **GitHub Pages** — ಉಚಿತ ಹೋಸ್ಟಿಂಗ್ ಸೇವೆ

<!--more-->

## Hugo ಎಂದರೇನು?

**Hugo** ಒಂದು ವೇಗವಾದ ಸ್ಥಿರ ತಾಣ ಜನರೇಟರ್. ಇದು Go ಭಾಷೆಯಲ್ಲಿ ಬರೆಯಲ್ಪಟ್ಟಿದೆ. ನೀವು Markdown ಫೈಲ್‌ಗಳಲ್ಲಿ ವಿಷಯ ಬರೆದರೆ, Hugo ಅದನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ HTML ಪುಟಗಳನ್ನಾಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ.

### Hugo ಯ ಪ್ರಯೋಜನಗಳು

- ಅತ್ಯಂತ ವೇಗವಾಗಿ ಪುಟಗಳನ್ನು ತಯಾರಿಸುತ್ತದೆ (ಸಾಮಾನ್ಯವಾಗಿ 100ms ಒಳಗೆ)
- ಡೇಟಾಬೇಸ್ ಅಗತ್ಯವಿಲ್ಲ
- ಸರ್ವರ್ ಭದ್ರತೆಯ ಸಮಸ್ಯೆಗಳಿಲ್ಲ
- ಹೋಸ್ಟಿಂಗ್ ವೆಚ್ಚ ಕಡಿಮೆ ಅಥವಾ ಶೂನ್ಯ

### ಫೈಲ್ ರಚನೆ

ನಮ್ಮ Hugo ತಾಣದ ಪ್ರಮುಖ ಫೋಲ್ಡರ್‌ಗಳು ಹೀಗಿವೆ:

| ಫೋಲ್ಡರ್ | ಉದ್ದೇಶ |
|---|---|
| `content/articles/` | ಎಲ್ಲಾ ಲೇಖನಗಳ ಮೂಲ ಫೈಲ್‌ಗಳು |
| `layouts/` | ಪುಟಗಳ ಟೆಂಪ್ಲೆಟ್‌ಗಳು |
| `assets/scss/` | ವೆಬ್‌ಸೈಟ್‌ನ ಶೈಲಿ (CSS) |
| `static/admin/` | CMS ಆಡಳಿತ ಪ್ಯಾನೆಲ್ ಫೈಲ್‌ಗಳು |
| `hugo.toml` | ತಾಣದ ಮುಖ್ಯ ಸಂರಚನೆ |

## Sveltia CMS ಎಂದರೇನು?

**Sveltia CMS** ಒಂದು ಮುಕ್ತ-ಮೂಲ (open-source) ವಿಷಯ ನಿರ್ವಹಣಾ ವ್ಯವಸ್ಥೆ. ಇದು ನಿಮ್ಮ ವೆಬ್‌ಸೈಟ್‌ನ `/admin/` ಮಾರ್ಗದಲ್ಲಿ ಬ್ರೌಸರ್‌ನಲ್ಲಿ ನೇರವಾಗಿ ತೆರೆದುಕೊಳ್ಳುತ್ತದೆ.

### CMS ನ ಕಾರ್ಯ ವಿಧಾನ

1. ಸಂಪಾದಕರು `/admin/` ಪುಟಕ್ಕೆ ಹೋಗಿ GitHub ಖಾತೆಯಿಂದ ಲಾಗಿನ್ ಆಗುತ್ತಾರೆ
2. Sveltia CMS ಒಂದು ಸರಳ ಸಂಪಾದನಾ ಇಂಟರ್‌ಫೇಸ್ ತೋರಿಸುತ್ತದೆ
3. ಲೇಖನ ಉಳಿಸಿದಾಗ, CMS ಸ್ವಯಂಚಾಲಿತವಾಗಿ GitHub ಗೆ Markdown ಫೈಲ್ ಕಳಿಸುತ್ತದೆ
4. GitHub Actions ಸ್ವಯಂಚಾಲಿತವಾಗಿ Hugo ಬಿಲ್ಡ್ ಚಾಲಿಸಿ GitHub Pages ನಲ್ಲಿ ಪ್ರಕಟಿಸುತ್ತದೆ

### CMS ನಲ್ಲಿ ಏನು ನಿರ್ವಹಿಸಬಹುದು?

ಪ್ರಸ್ತುತ CMS ನಲ್ಲಿ ಕೇವಲ **ಲೇಖನಗಳನ್ನು (Articles)** ಮಾತ್ರ ನಿರ್ವಹಿಸಬಹುದು:

- ಹೊಸ ಲೇಖನ ರಚಿಸುವುದು
- ಲೇಖನ ಸಂಪಾದಿಸುವುದು
- ಲೇಖನ ಪ್ರಕಟಿಸುವುದು ಅಥವಾ ಡ್ರಾಫ್ಟ್ ಆಗಿ ಇಡುವುದು
- ಚಿತ್ರಗಳನ್ನು ಅಪ್‌ಲೋಡ್ ಮಾಡುವುದು

## GitHub Pages ಹೋಸ್ಟಿಂಗ್

ನಮ್ಮ ವೆಬ್‌ಸೈಟ್ ಅನ್ನು **GitHub Pages** ಮೂಲಕ ಉಚಿತವಾಗಿ ಆನ್‌ಲೈನ್‌ನಲ್ಲಿ ಇಡಲಾಗಿದೆ.

### ಪ್ರಕಟೀಕರಣ ಪ್ರಕ್ರಿಯೆ

```
ಲೇಖಕರು CMS ನಲ್ಲಿ ಬರೆಯುತ್ತಾರೆ
        ↓
GitHub ಗೆ Markdown ಉಳಿಯುತ್ತದೆ
        ↓
GitHub Actions Hugo ಬಿಲ್ಡ್ ಚಾಲಿಸುತ್ತದೆ
        ↓
HTML ಪುಟಗಳು GitHub Pages ನಲ್ಲಿ ಪ್ರಕಟವಾಗುತ್ತವೆ
```

ಈ ಪ್ರಕ್ರಿಯೆ ಸಂಪೂರ್ಣ ಸ್ವಯಂಚಾಲಿತ — ಒಮ್ಮೆ ಲೇಖನ ಉಳಿಸಿದರೆ ಕೆಲವು ನಿಮಿಷಗಳಲ್ಲಿ ವೆಬ್‌ಸೈಟ್‌ನಲ್ಲಿ ಕಾಣಿಸಿಕೊಳ್ಳುತ್ತದೆ.

## ಸಾರಾಂಶ

ನಮ್ಮ CMS ವ್ಯವಸ್ಥೆ ಸರಳ, ವೇಗವಾದ ಮತ್ತು ಉಚಿತ ಸಾಧನಗಳ ಮೇಲೆ ಆಧಾರಿತವಾಗಿದೆ:

- **Hugo** ವಿಷಯವನ್ನು ವೆಬ್‌ಸೈಟ್ ಆಗಿ ಪರಿವರ್ತಿಸುತ್ತದೆ
- **Sveltia CMS** ಸಂಪಾದಿಸಲು ಸುಲಭ ಇಂಟರ್‌ಫೇಸ್ ನೀಡುತ್ತದೆ
- **GitHub** ವಿಷಯ ಸಂಗ್ರಹ ಮತ್ತು ಸ್ವಯಂ ಪ್ರಕಟೀಕರಣ ನಿರ್ವಹಿಸುತ್ತದೆ
- **GitHub Pages** ಉಚಿತ ಹೋಸ್ಟಿಂಗ್ ಒದಗಿಸುತ್ತದೆ

ಈ ಸಂಯೋಜನೆ ತಾಂತ್ರಿಕ ಜ್ಞಾನವಿಲ್ಲದ ಬರಹಗಾರರಿಗೂ ಸುಲಭವಾಗಿ ಬಳಸಬಲ್ಲ ಪ್ರಕಟೀಕರಣ ವ್ಯವಸ್ಥೆಯನ್ನು ನೀಡುತ್ತದೆ.
